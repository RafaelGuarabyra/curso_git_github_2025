git init . 		= Cria um repo git na pasta atual
git status 		= Mostra o status do trabalho
git add 		= Adiciona arquivo a ser commitado
git commit -m 		= Faz o commit do que estava para ser commitado
git log 		= Mostra todo o histórico de commit

Revisão sobre o Fluxo de Trabalho do Git:
1.	Criar o diretório do projeto
2.	Iniciar o git (git init .)
3.	Edita ou cria arquivos
4.	Git status
5.	Git add <nome_arquivo>
6.	Git status
7.	Git commit -m “mensagem”
8.	Git status
9.	Volta para o passo 3
